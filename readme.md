WDIO Slick Reporter
==================

> A slick WebdriverIO reporter intended for local development.

## Installation

Install `wdio-slick-reporter` as a devDependency.

```bash
npm install --save-dev wdio-slick-reporter
```

## Configuration

Use like any other reporter:

```js
// wdio.conf.js
module.exports = {
  // ...
  reporters: ['slick'],
  // ...
};
```

## Development

Run tests:

```bash
$ npm test
```
